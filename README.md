# 🐍 GestureSnake

GestureSnake 是一个基于 **MediaPipe Hands** 的手势控制贪吃蛇应用。  
系统通过摄像头实时识别用户手部动作，将自然手势映射为游戏控制指令，实现**无需键盘的手势交互游戏体验**。

本项目为应用型课程设计项目，强调完整性、稳定性与用户体验。

---

## ✨ Features

- 🎥 基于摄像头的实时手势识别（MediaPipe Hands）
- 🖐️ 自然手势控制贪吃蛇游戏
- 🔁 手势防抖与冷却机制，减少误触发
- 🧠 手势识别与游戏逻辑解耦，结构清晰
- 🖥️ 本地运行，无需联网

---

## 🎮 Gesture Mapping

| 手势 | 功能 |
|----|----|
| 食指指向（左 / 右 / 上） | 控制蛇的移动方向 |
| V 手势（食指 + 中指） | 向下移动 |
| OK 捏合手势 | 暂停 / 继续游戏 |
| 五指张开 | 重新开始游戏 |

> 说明：  
> 由于“指向下方”手势在实际使用中稳定性较差，本项目采用 **V 手势** 作为向下移动指令，以提升识别准确率和用户体验。
